# Tutoriales de programación con LearnPack
<!--endhide-->

En 4Geeks.com, verás la palabra "LearnPack" aquí y allá. Usarás LearnPack MUCHO para practicar tutoriales de programación y pequeños ejercicios. Este artículo explica qué es y cómo usarlo para que puedas ejecutar todos los tutoriales sin problemas.

## ¿Qué es LearnPack?

LearnPack es una herramienta diseñada por 4Geeks para enseñar y aprender habilidades relacionadas con la programación mediante tutoriales y ejercicios interactivos, así como auto-calificados y habilitados con IA. Actúa como un gestor de paquetes para contenido educativo, similar a plataformas como npm, brew, etc., pero exclusivamente para paquetes educativos técnicos.

### ¿Qué hace que LearnPack sea tan único?

En lugar de ver un video o leer un artículo, LearnPack presenta pequeños desafíos de forma incremental; cada uno está auto-calificado para retroalimentación instantánea y una solución de video tutorial (a veces).

> Esta es la mejor manera de aprender habilidades relacionadas con la programación.
